Welcome to the Flask API for Code Execution

This API allows you to execute Python code securely via POST requests. Below are the details of how to use this API effectively.

API Endpoint

Send a POST request to the following endpoint:

POST /execute

Request Format

The API expects a JSON payload containing the Python code to execute. Example:

    
                {
                    "code": "print('Hello, World!')"
                }

Response Format

The response will be a JSON object containing the output or error messages from the executed code:


            {
                "output": "Hello, World!\n"
            }

If there is an error in the code:

            {
                "success": false,
                "error": "SyntaxError: unexpected EOF while parsing"
            }

Example Usage

You can test this API using tools like Postman or with a Python script using the requests library:

                import requests
    
                url = "http://127.0.0.1:5000/execute"
                payload = {
                    "code": "print('Hello, Flask!')"
                }
                response = requests.post(url, json=payload)
    
                if response.status_code == 200:
                    print("Response:", response.json())
                else:
                    print("Failed to connect:", response.status_code)

Notes

  • Ensure the Python code you send is safe and does not include malicious commands.
